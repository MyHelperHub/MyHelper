## ✨ 功能特性

### 🌍 主页展示

![主页展示](https://tc.z.wiki/autoupload/f/QMrmJUtVXl8-V_8WyW9oE5mesdO83n0jJRcmVXjsIsc/20251102/uVTX/250X420/image.png)

### 🧩 插件系统

插件市场提供丰富的扩展功能，满足不同场景的使用需求：

- **一键安装管理** - 从插件市场快速安装和管理插件
- **轻量级设计** - 每个插件仅占用约 1MB 空间
- **无限扩展** - 通过插件系统实现功能定制
- **开发友好** - 支持开发者自行开发插件（详见插件开发文档）

![插件市场](https://tc.z.wiki/autoupload/20250603/7B8A/1300X700/image.png)

### 🎨 主题系统

完善的主题定制系统，打造专属个性界面：

- **深色/浅色模式** - 自动适配系统主题或手动切换
- **预设主题** - 多款精心设计的预设主题可选
- **自定义主题** - 支持自定义颜色、透明度、圆角等细节
- **毛玻璃效果** - 现代化的半透明窗口效果
- **主题变量系统** - 完整的主题变量体系，所有UI元素自动适配

![主题设置](https://tc.z.wiki/autoupload/f/QMrmJUtVXl8-V_8WyW9oE5mesdO83n0jJRcmVXjsIsc/20251102/Dc0m/1137X770/image.png)

### 🐾 桌面宠物

Live2D 萌宠陪伴你的工作时光：

- **替代头像显示** - 可用可爱的 Live2D 宠物替代传统头像
- **预置模型** - 内置多款精选 Live2D 模型
- **自定义导入** - 支持导入自己喜欢的 Live2D 模型（Cubism 2.1/3.x/4.x）
- **互动动画** - 点击触发随机动作和表情
- **灵活配置** - 可自定义宠物大小、开关显示

告别繁琐的应用查找，一键直达常用应用和网站：

- 快速启动本地应用程序
- 收藏常用网站快捷访问
- 自定义图标和分组
- 智能搜索定位

随时记录灵感和待办事项：

- 支持置顶和自由拖拽

### ⌨️ 快捷输入

提升文本输入效率，告别重复劳动：

- **剪贴板历史** - 自动记录复制历史，随时调用
- **常用文本** - 保存常用文本模板，一键粘贴
- **智能粘贴** - 选择即自动复制并粘贴到当前应用
- **双栏切换** - 剪贴板历史与常用文本快速切换

## 🧩 为开发者准备

如果你是开发者，MyHelper欢迎你参与：

- **Vue3 + TypeScript** 开发环境，热重载秒级生效
- 默认集成**PrimeVue组件库**，也可使用任意Vue3组件库
- 支持**窗口自定义**、浏览器API调用、配置存储
- 轻松打包上传，快速审核上线

> **插件开发说明**：由于目前暂未探寻出合适的**社区插件审核方案**，故暂且禁止第三方上传插件，若您有好的插件想法或方案欢迎与作者沟通交流。

详细文档请参考 [插件开发指南](./docs/plugin-development.md)
