# 插件开发步骤
1.拉取项目代码
2.在根目录下执行`npm install`
3.在`plugin`下的`my-plugin`文件夹下执行`npm install`
4.填写`my-plugin`文件夹下的`myPlugin.json`文件
5.运行`npm run dev`即可编写插件
